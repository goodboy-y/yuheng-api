# 玉衡API

将sql直接转换为api接口，支持mysql, oracle, postgresql, sqlite等等(使用JPA，JPA支持的数据库都支持)

技术选型

后端

- springboot4
- jpa

前端

- vue3
- element-plus
- vite

## 环境要求

JDK17+
