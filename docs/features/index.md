# 功能概览

玉衡API 提供以下核心功能模块：

| 模块         | 说明                                         |
| ------------ | -------------------------------------------- |
| 数据源管理   | 管理多个数据库连接，支持 MySQL 和 Oracle      |
| API 配置     | 将 SQL 查询配置为 RESTful API 接口            |
| 客户端管理   | 管理 API 调用方，生成 ClientId/Secret 凭证    |
| 插件系统     | 可扩展的插件机制，内置分页插件                |
| 字段映射     | 数据库字段与 API 返回字段的映射和别名          |
| 访问控制     | 控制客户端对 API 的访问权限                   |
| 数据导出     | 将 API 查询结果导出为 Excel 文件              |
| 调用日志     | 记录 API 调用日志，支持归档                   |

## 架构分层

```
┌─────────────────────────────────────────┐
│              管理端 UI                    │
├─────────────────────────────────────────┤
│          Controller 层 (REST API)        │
├─────────────────────────────────────────┤
│           Filter 层 (鉴权/日志)          │
├─────────────────────────────────────────┤
│            Service 层 (业务逻辑)          │
├─────────────────────────────────────────┤
│           Plugin 层 (可扩展处理)          │
├─────────────────────────────────────────┤
│   Repository 层 (JPA 数据访问)            │
├─────────────────────────────────────────┤
│        外部数据源 (MySQL/Oracle)          │
└─────────────────────────────────────────┘
```
